Detailed Description

Intensity and count metrics for each base.

The corrected intensity metrics provide the percent base plot and the intensity plots.

For 4-dye systems, the intensity corrected for cross-talk between the color channels and phasing and prephasing. These are used in base calling. Note, the software generates phasing/prephasing estimates after cycle25 (for most platforms).

For 2-dye systems, the values are calculated is another way and aid in accessing the progress of the run.

Note
Supported versions: 2 and 3
